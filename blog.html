1. What is DOM and what is the purpose of DOM?

DOM means Document Object Model. This represent a document with logical tree. Each branch of trees represent as a node. Using DOM we can easily access HTML elements and can manipulate or changes in styles. DOM mainly used to access different HTML elements and changes their inner value or inner text or their styles.

2. How will you select HTML elements using DOM? Name the DOM methods?

We can use DOM in many ways. Their are several ways to select HTML elements:

  • getElementById
  • getElementsByClassName
  • querySelector
  • querySelectorAll
  • getElementByTagName

3. What is event bubble?

Event bubble is one of the most important concept of DOM. This happen when an element receives an event, and then bubbles up to its parent and ancestor element in the DOM tree until it gets to the root element. Element can receive different types of events like click, blur, scroll and so on. After trigger an event that event stars to propagate and this is the default behavior of an event.

4. What is a callback function and why will you use it?

Callback function is a function that passed into another function as an argument. Callback function executed after another function has finished execution.
The main purpose is to execute code in response to an event, this might be user-initiated. such as, mouse click, mouse hover, typing and so on.
